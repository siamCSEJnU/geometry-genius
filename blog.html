3.What is event bubble?

-Event bubbles happens when an element gets triggered by an event, and that event bubbles up or propagated to its parent elements in the DOM tree until it gets to the root element.

What is a callback function and why will you use it?

-When a function is passed as an argument to another function then it is called callback function.Mainly it is used to ensure that the target function will not start to execute until a specific piece of code belonging to function is not executed.
